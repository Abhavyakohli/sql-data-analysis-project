# Sales Data Analysis Project

## Business Understanding
The objective of this project is to analyze sales data to identify trends, top-performing products, and revenue growth patterns to support business decision-making.

## Data Understanding
The dataset contains sales records with information such as order date, product name, quantity sold, price, and total revenue.

## Technologies Used
- SQL
- Python
- Pandas
- Power BI
- Excel

## Approach
1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. SQL queries for insights  
4. Data visualization using Power BI  

## Key Insights
- Identified top-selling products
- Monthly revenue trends
- Regional performance comparison
